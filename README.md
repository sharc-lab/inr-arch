# sharc-INR

This project aims to combine hardware acceleration and software modeling for implicit neural representations, also known as INRs.

This project is still in development so stay tuned!

## HW Library

- Activations
  - ELU
  - Hardtanh
  - LeakyReLU
  - ReLU
  - GELU
  - Sigmoid
  - SiLU
  - Tanh
  - Softsign
  - Sin
  - Cos
- Linear
  - Linear
- Normalization
  - LayerNorm

## Model Architectures

- SIREN
